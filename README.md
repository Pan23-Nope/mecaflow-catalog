# MecaFlow - Mechanical Supply Chain Management

A premium, localized supply chain management system for mechanical engineering firms. Featuring a high-performance product catalog and a secure admin dashboard.

## 🚀 Live Access
- **Product Catalog**: [index.html](https://Pan23-Nope.github.io/mecaflow-catalog/index.html)
- **Admin Dashboard**: [admin.html](https://Pan23-Nope.github.io/mecaflow-catalog/admin.html)

- **Username**: `admin`
- **Password**: `hiphop-never-die`

## ✨ Core Features
- **Premium UI/UX**: Dark-themed interface with glassmorphism and smooth transitions.
- **Product Catalog**: Dynamic filtering by customer, image zoom/lightbox, and detailed specifications.
- **Customer Management**: Full CRUD operations for managing business partners.
- **Admin Portal**: Integrated product management with image upload previews.
- **Data Persistence**: Uses `localStorage` with a ready-to-deploy Supabase integration.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3, Tailwind CSS.
- **Interactions**: Vanilla JavaScript (ES6+).
- **Backend Architecture**: Supabase (PostgreSQL) compatible.

## 📦 Deployment
The project is optimized for GitHub Pages. 

### Local Development
1. Clone the repository.
2. Open `index.html` in your browser (Live Server recommended).
3. Access the admin panel via `admin.html`.

### Scripts
- `deploy.ps1`: PowerShell script for automated deployment (Windows).
- `deploy.sh`: Shell script for automated deployment (Linux/Mac).
