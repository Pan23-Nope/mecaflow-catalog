# MecaFlow Catalog

Mechanical parts catalog with Supabase integration and admin portal.

## Features
- Premium industrial design
- Dynamic product catalog
- Admin portal with CRUD operations
- Responsive layout

## Deployment
Deployed via GitHub Pages.

### Auto Commit & Deploy
We provide scripts to automate the staging, committing, and pushing process:

**Windows (PowerShell):**
```powershell
./deploy.ps1 "Your commit message"
```

**Linux/macOS/Git Bash:**
```bash
chmod +x deploy.sh
./deploy.sh "Your commit message"
```
